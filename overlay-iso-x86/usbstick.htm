Installing SystemRescueCd on an USB stick

The default version of systemRescueCd aims to work from a CDRom. It's also possible to install SystemRescueCd on a USB stick.

You just need a 512 MB stick, and you can install it from Linux or windows. You can find instructions about how to install in the official SystemRescueCd website, in the "Online manual" section.
